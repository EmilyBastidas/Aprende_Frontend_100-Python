<h1>Proyecto: Tienda de Lentes (Frontend)</h1>

Estoy construyendo el frontend de una página de venta de lentes con el objetivo de aprender ReactPy y practicar el uso de esta librería. Para ello, estoy utilizando:

- ReactPy

- FastAPI

-Bootstrap

ReactPy es una librería para construir interfaces web usando Python (puedes encontrarla aquí: reactpy.dev).

<h3>Estructura del proyecto</h3>

- venv/
- main.py
- README.md
- requirements.txt

<h3>¿Qué he hecho hasta ahora?</h3>

Hasta el momento, he completado el Paso 1, que consiste en construir el Navbar de la página utilizando ReactPy y Bootstrap.

<h3>Pasos para ejecutar el proyecto</h3>
<li>Activar el entorno virtual</li>

En Mac:
source venv/bin/activate

En Windows:
venv\Scripts\activate

<li>Instalar dependencias</li>
pip install -r requirements.txt

<li>Ejecutar la aplicación</li>
uvicorn main:app

<li>Abrir la app en el navegador</li>

## Notas de aprendizaje

Este proyecto me está ayudando a aprender y practicar ReactPy junto con FastAPI y Bootstrap. A continuación, dejo algunas notas sobre lo que he aprendido hasta ahora:

Conceptos que estoy practicando

ReactPy
Me permite construir interfaces web usando componentes en Python, similar a React pero sin usar JavaScript.

FastAPI
Se utiliza como servidor backend para servir la aplicación y renderizar los componentes de ReactPy.

Bootstrap
Se usa para estilizar rápidamente la interfaz y crear un diseño responsivo.

Mi recomendación, siempre pero siempre, busca la documentación y basate en ella.
